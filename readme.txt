【事前準備】
同ディレクトリ内の「rss-reader.bat」のショートカットを作成してデスクトップに配置する。
バッチは以下の内容
・サーバ「cgiserver.py」の起動
・Webブラウザの新規タブにてRSSReaderのページ（http://localhost:8000/index.html）を開く

【実施内容】
デスクトップにある「rss-reader.bat」をダブルクリックするとアプリが起動するので、一通り触ってもらう。
その後、実際にソースを修正して動きが変わることを体験してもらう。
（担当者用PCにも同じ環境が整っているので、見せながら実施してあげるイメージ）

【課題】
●機能拡張的な内容
　1. cgi-bin/rss-reader.pyのプルダウンを定義している箇所に以下の項目を追加して上書き保存する。
　　・サイエンス
　　・ソフトウェア
　　※「<option value=""></option>」を直下にコピペして他の項目を参考に値を設定させる
　2. ページを更新するとプルダウンにカテゴリが追加され、対応した情報が新たに取得できるようになる。 
　
●（予備）文字化け対策について
　※視覚的に変化が分かりやすいですが、取り扱うのが難しそうならスキップしてください。
　・18行目にある「sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='UTF-8')」を削除してみる
　→標準出力の文字コードのままだと文字化けしてしまう
